Detailed Description

Contains the SpeechBubbleController class which handles all incoming requests from the frontend.

This file defines the SpeechBubbleController class, which is responsible for handling incoming requests related to speech bubbles from the frontend. It provides methods to update existing speech bubbles and parse frontend JSON data to backend-compatible SpeechBubbles.
