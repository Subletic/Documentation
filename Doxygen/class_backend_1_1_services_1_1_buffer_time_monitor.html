Detailed Description

Service for monitoring the time of SpeechBubbles in the list.

The BufferTimeMonitor class monitors the time of the oldest SpeechBubble in the list. If a SpeechBubble exceeds the time limit, it is removed from the list and added to the timedOutSpeechBubbles list. The class also exports the timed-out SpeechBubble as a webvtt file.
