Detailed Description

Service for managing the audio buffers to be sent to the frontend.

The FrontendAudioQueueService class is a concurrent queue of short arrays representing audio buffers. It ensures that the enqueued buffers have the correct type (signed 16-bit) and size (48000 samples).
