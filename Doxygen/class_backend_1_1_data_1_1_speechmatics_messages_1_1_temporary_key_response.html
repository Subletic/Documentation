Detailed Description

Speechmatics API response: Temporary Key (no official name)

Direction: Server -> Client When: After a Temporary Key request for On-demand customers has been requested Purpose: Give a temporary key for the Speechmatics RT API, valid until

  • the configured time-to-live for the key has been reached, or
  • the customer has reached their monthly limit for the RT API Effects: n/a
