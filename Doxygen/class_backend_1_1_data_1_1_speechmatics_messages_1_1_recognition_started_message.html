Detailed Description

Speechmatics RT API message: RecognitionStarted.

Direction: Server -> Client When: After StartRecognitionMessage Purpose: Signal that Server has started the transcription, and inform about the settings and details Effects:

  • Transcription has begun
  • AddAudio messages and EndOfStreamMessages will now be accepted
